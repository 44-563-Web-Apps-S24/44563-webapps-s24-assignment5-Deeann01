Reason for Breaking : When you shrink the browser window, you'll notice that the row with the drink names breaks first. This happens because the boxes holding the drink names are set to adjust their size based on medium-sized screens. As the window gets smaller, these boxes try to stay a certain size, so they end up stacking on top of each other sooner than the row with the letters. The row with the letters, on the other hand, keeps its size fixed even as the window shrinks because it's set to be a certain width for small screens. So, it stays together longer before breaking.
